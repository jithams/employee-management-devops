# Phase 4 - Commands Used

## Repository Management

git remote -v

git status

tree -L 2

## Git Branching

git branch

git checkout -b feature/readme-update

git checkout main

git merge feature/readme-update

git branch -d feature/readme-update

## Collaboration

git push origin feature/github-collaboration

git pull origin main

## Conflict Resolution

git checkout -b feature/conflict-demo

git merge feature/conflict-demo

git add README.md

git commit -m "Resolved merge conflict in README"

git push origin main

## Repository Maintenance

git tag v1.0

git tag

git push origin v1.0

git add .gitignore

git commit -m "Added .gitignore file"

git push origin main
