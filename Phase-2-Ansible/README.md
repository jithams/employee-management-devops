# Phase 2 - Advanced Linux Administration (RHCE)

## Objective

The objective of this phase is to automate Linux server administration using Ansible by implementing infrastructure automation, role-based automation, configuration management, automated application deployment, and server provisioning.

---

## Technologies Used

- AWS EC2
- Amazon Linux 2023
- Ansible
- SSH
- Apache (httpd)
- Git
- GitHub

---

## Tasks Completed

### Step 1 - Ansible Installation and Configuration
- Installed Ansible
- Configured Inventory
- Configured Passwordless SSH
- Configured ansible.cfg

### Step 2 - Infrastructure Automation
- Ping Playbook
- User Management Playbook
- Apache Installation Playbook
- Apache Service Management

### Step 3 - Role-Based Automation
- Created Apache Role
- Executed Role using Playbook

### Step 4 - Configuration Management
- Managed index.html using Ansible
- Updated Web Page
- Verified Configuration

### Step 5 - Automated Application Deployment
- Created Employee Management Portal
- Automated Deployment using Ansible Role
- Verified Application

### Step 6 - Server Provisioning
- Created Provisioning Playbook
- Provisioned Server using Apache Role
- Verified Provisioning
