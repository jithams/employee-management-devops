# Phase 1 - Linux Administration (RHCSA)

## Objective

The objective of this phase is to build a strong foundation in Linux system administration by performing real-world administrative tasks on an AWS EC2 instance running Amazon Linux 2023.

## Topics Covered

- Linux Installation and Configuration
- User and Group Management
- File Permissions and ACL
- Package Management
- Service Management
- Storage Management
- SSH Configuration
- Firewall Configuration
- Shell Scripting
- Log Management and Troubleshooting

## Environment

- AWS EC2
- Amazon Linux 2023
- GitHub
- Git
- SSH
- XFS File System

## Tasks Completed

- Created Linux users and groups
- Configured file permissions and ACLs
- Installed and managed software packages
- Managed Linux services using systemctl
- Added and mounted an EBS volume
- Configured persistent storage using /etc/fstab
- Verified SSH configuration
- Configured firewalld
- Developed a shell backup script
- Performed log analysis and system monitoring

## Learning Outcomes

- Linux system administration
- User management
- Storage management
- Firewall administration
- Shell scripting
- Troubleshooting
- GitHub project documentation
